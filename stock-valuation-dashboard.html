Overview

This dashboard delivers an equity valuation workflow inside a single Excel workbook — from raw financial statement data through to a weighted intrinsic value consensus and a BUY / SELL signal.

It was built to support retail investment research on US equities, replacing ad-hoc spreadsheets with a repeatable, institutionally structured process. Any ticker available on StockAnalysis.com can be loaded through the Power Query pipeline without rebuilding the model — reducing a full valuation from hours to minutes.

Annual basis by default. Convertible to quarterly or trailing-twelve-months (TTM) view without structural changes to the data model. Dashboard language reflects the primary investment market (English / USD).

Data & Methodology

Financial data is sourced entirely from StockAnalysis.com Pro, covering 10 years of annual statements across income, balance sheet, cash flow, and ratios. Price and market data is pulled from Yahoo Finance, and company profile metadata from the Stock Data add-in for Excel. All sources are integrated through a unified Power Query pipeline.

Business Value

Built for retail investors who want institutional-grade rigor without institutional overhead. The workbook consolidates DCF, Graham's, comparable multiples, and historical multiples into a single file — reducing a full valuation from hours to minutes and eliminating the scattered, ad-hoc spreadsheets that make research hard to reproduce or revisit.

Loading a new ticker requires only a Power Query refresh — no structural rebuild. The color-coded input convention (blue = manual, black = formula, green = cross-sheet link) keeps the model auditable and easy to hand off. Any US equity available on StockAnalysis.com can be analyzed with the same workflow.
